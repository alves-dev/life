## Descrição

A ideia deste projeto é construir um eco sistema de dados e informações referente as minhas atividades diárias; utilizando as mais diversas ferramentas e aplicações diferentes. Idealmente quero utilizar frameworks e linguagens diferentes; é provável que algumas escolhas não façam sentidos do ponto de vista de uma arquitetura ideal, porem o objetivo maior aqui é aprender novas coisas.

## Uso

Este projeto utiliza diversas tecnologias e frameworks para criar um ecossistema de informações. Nenhum exemplo de utilização específico foi providenciado.

## Estrutura do Projeto

* `src/`: Contém o código-fonte do projeto.
* `public/`: Contém arquivos estáticos, como imagens e arquivos HTML.
* `package.json`: Arquivo que contém informações sobre o projeto e suas dependências.

## Notas

Este projeto é um ambiente de aprendizado e pode conter decisões arquitetônicas não ideais.
