## Descrição

A ideia deste projeto é construir um eco sistema de dados e informações referente as minhas atividades diárias; utilizando as mais diversas ferramentas e aplicações diferentes. Idealmente quero utilizar frameworks e linguagens diferentes; é provável que algumas escolhas não façam sentidos do ponto de vista de uma arquitetura ideal, porem o objetivo maior aqui é aprender novas coisas.
